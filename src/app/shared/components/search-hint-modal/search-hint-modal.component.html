Search Help
  1. Search by Name:

    You can type a user's full or partial name and the results will propagate accordingly

  2. Search by Post/Topic Title:

    You can type a topic/post title into search field and the results will propagate accordingly

  3. Search by creation date:

    You will need to use the following syntax to get proper results for date based search.

    1. Date only: Please enter your date in the format DD-MM-YYYY like 18-06-2020 for 18th June 2020.

    2. Date with Time: Please enter your date in 24-hours format DD-MM-YYYY HH:mm like 18-06-2020 20:28 for 18th June 2020, 08:28PM.
